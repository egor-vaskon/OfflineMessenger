Help page

Offline Messenger is the simple social app that working with WIFI-DIRECT.

Advices



This app needs wifi to work.
If app is not working check if wifi is turned on.

If you don't see devices in the list, disable and then enable wifi and wait.

Creating chat


1. Open start page of this application.
2. Click to the create chat button.
3. Wait for other people to join.

Joining chat


1. Open start page of this app.
2. Wait few seconds.
3. Open chat you want to join.
4. Wait few seconds.
5. You can't join a chat, which have two or more members.

Sending messages


1. This app supports text messages and text messages with one image.
2. To send a message press the button with airplane.
3. Images are located in the gallery.

How to change the user name


1. Open start page of this app.
2. You will see input filed.
3. Enter username in this field.
4. Your username will be changed.
